Features

Everything you need.
Nothing you don't.

Real-time visibility into every AI coding session — from a single system tray app.

Live Session Monitoring

See all active AI coding sessions with their current status at a glance. Know what's running, what's waiting, and what just finished.

Tool Activity Tracking

Real-time updates on which tools are executing — file edits, bash commands, searches, and more — with extracted context like file paths and patterns.

System Tray Integration

Lives in your system tray with a status-aware icon. Green when sessions are active, yellow when waiting, gray when idle. Click to toggle the dashboard.

One-Click Setup

Install hooks with a single click. Jackdaw configures everything automatically — no manual config files or terminal commands needed.

How It Works

Three steps. That's it.

From install to full visibility in under a minute.

1

Install Jackdaw

Download the app for your platform. No package managers, no dependencies.

2

Install hooks

Click one button. Jackdaw configures your AI coding tools to send events automatically.

3

Start coding

Open your AI coding tool. Sessions appear in the dashboard in real time.

Install

One command. Done.

Detects your OS and architecture, grabs the latest release, and drops the binary in your path.

$ curl -fsSL https://jackdaw.whattheflock.dev/install.sh | sh

Or grab a build directly from the releases page.
