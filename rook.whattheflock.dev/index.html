Features

Everything you need.
Nothing you don't.

Zero-config local development with SSL, runtimes, and services — managed from a single desktop app.

Automatic SSL

Local HTTPS via mkcert with zero configuration. Every site gets a trusted certificate automatically.

PHP Management

Multiple PHP versions with per-site FPM pools. Switch versions per project without touching your system install.

Database Services

MySQL, PostgreSQL, and Redis in one click. Start, stop, and manage services without Docker or manual installs.

Plugin Architecture

Extensible to any language stack. Node.js support built-in, with an open API for community plugins.
